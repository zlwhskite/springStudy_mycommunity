## 기능
- 게시글 CRUD
- 페이징
- 회원가입, 찾기
- 로그인, 로그아웃
- 관리자 페이지
- 댓글
- 조회 수
- 검색
- 출석체크(1,2,3,X,1,2...)
- 게시글 정렬(최신순, 인기순)
